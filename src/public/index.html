Keyboard Shortcuts

  • Toggle Mouse Pass-through:
    Press Control+` to allow mouse clicks to "pass through" this window to the applications behind it. Press the key combination again to restore normal mouse interaction.
  • Minimize Window Content:
    Press Control+Alt+Z to minimize the content section of the application. Press the key combination again to restore content section size.
  • Resize Window:
    Press Control + Arrow Keys to resize the application window.
  • Move Window:
    Press Control + Alt + Arrow Keys to move the application window.
